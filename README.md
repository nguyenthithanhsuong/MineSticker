# MineSticker
Minesweeper but with the dark lord.

## Structure
- client: React + Vite
- server: NestJS API
- shared: Game domain classes (Character, Tile, Face, Matrix)

## Quick start
1) Install dependencies
	- client: npm install
	- server: npm install

2) Run server
	- npm run start:dev

3) Run client
	- npm run dev

The API is available at http://localhost:3000/state and the UI at http://localhost:5173.
